Denew Backend API

Welcome to the Denew Backend API. This is the API server for the Denew networking platform.

API Documentation

The following endpoints are available:

Authentication

POST /api/register/ - Register a new user

POST /api/login/ - Login and get authentication tokens

POST /api/logout/ - Logout and invalidate tokens

User Profile

GET /api/profile/ - Get user profile information

PUT /api/profile/update/ - Update user profile

Tasks

GET /api/tasks/ - List all tasks

GET /api/tasks/current/ - Get current task

POST /api/tasks/start-set/ - Start a task set

POST /api/tasks/start/ - Start a task

POST /api/tasks/complete/ - Complete a task

Transactions

POST /api/deposit/ - Make a deposit

POST /api/withdrawal/ - Request a withdrawal

GET /api/transactions/ - Get transaction history

For more information, please refer to the API documentation or contact support.
